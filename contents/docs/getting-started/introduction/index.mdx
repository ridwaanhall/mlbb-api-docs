This project provides an API for fetching various analytics and data related to heroes 
in the game Mobile Legends: Bang Bang (MLBB). The API includes endpoints for hero rankings, 
positions, details, skill combinations, ratings, relationships, counter information,
 and compatibility.

<Note title="Remember" type="danger">
  Please remember that this API is not officially recognized by MLBB and is not affiliated with them.
</Note>

## Conclusion

This project is designed to fetch APIs from the game Mobile Legends: Bang Bang and display them.
It provides a comprehensive set of endpoints to retrieve various data and analytics related to 
the heroes in the game. The API includes features such as hero rankings, positions, 
detailed statistics, skill combinations, ratings, relationships, counter information, and 
compatibility. With this project, you can easily access and utilize the rich data available for 
Mobile Legends: Bang Bang to enhance your applications or analyses.
