This section provides an overview of the MLBB API's multi-language support. Currently, the API supports 3 languages:

```txt
- English (default)
- Russian
- Indonesian
```

<Note title="Remember" type="danger">
  English will be used automatically if no language parameter is specified.
</Note>

## Getting Started

<Note title="Attention" type="warning">
  If you encounter any bugs, incompatible URLs, or have feature requests or other inquiries, please submit a pull request on my GitHub by clicking the GitHub link in the navbar.
</Note>

To begin using another language, add this parameter into our API.

```txt
https://mlbb-stats.ridwaanhall.com/api/hero-rank/?days=7&rank=mythic&size=10&index=2&sort_field=pick_rate&sort_order=asc&lang=ru # for russian language
https://mlbb-stats.ridwaanhall.com/api/hero-rank/?days=7&rank=mythic&size=10&index=2&sort_field=pick_rate&sort_order=asc&lang=id # for indonesian language
```
