---
title: GET Hero Compatibility
description: API endpoint uses main_heroid (1-129) to fetch hero compatibility data.
---

## Endpoint

```txt
GET /api/hero-compatibility/<int:main_heroid>/
```

## Path Parameters

1. **main_heroid**
   The ID of the main hero whose compatibility information is to be fetched.

   - **Possible Values:** From `1` to `129`
   - **Type:** integer
   - **Required:** `True`

## Example Requests

```txt
GET /api/hero-compatibility/127/
```

## Example Usage

### Python requests

    ```python {13} showLineNumbers
    import requests
    import json

    def fetch_api(api_url):
        try:
            response = requests.get(api_url)
            response.raise_for_status()
            return response.json()
        except requests.exceptions.RequestException as e:
            print(f"Error: {e}")
            return None

    api_url = "https://mlbb-stats.ridwaanhall.com/api/hero-compatibility/127/"
    data = fetch_api(api_url)

    print(json.dumps(data, indent=4) if data else "No data fetched.")
    ```
