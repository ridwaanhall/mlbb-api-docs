---
title: GET MPLID Team Detail
description: API endpoint to get team details including logo, name, social media, and roster.
---

## Endpoint

```txt
GET /api/mplid/teams/rrq/
```

## Path Parameters

1. **team_id**
   The ID of the hero whose details are to be fetched.

   - **Possible Values:** `rrq`, `geek`, `onic`, `btr`, `ae`, `tlid`, `evos`, `dewa`, `navi`.
   - **Type:** string
   - **Required:** `True`

## Example Requests

```txt
GET /api/mplid/teams/rrq/
```

## Example Usage

### Python requests

    ```python {13} showLineNumbers
    import requests
    import json

    def fetch_api(api_url):
        try:
            response = requests.get(api_url)
            response.raise_for_status()
            return response.json()
        except requests.exceptions.RequestException as e:
            print(f"Error: {e}")
            return None

    api_url = "https://mlbb-stats.ridwaanhall.com/api/mplid/teams/rrq/"
    data = fetch_api(api_url)

    print(json.dumps(data, indent=4) if data else "No data fetched.")
    ```
